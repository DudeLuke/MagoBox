Here you can put all information related to FDG, allowing for automatic patching based on what entities and blocks are in that level as well as a level's background and tileset. The triple minus mark signifies the start of a new FDG list, and the text afterwards is the "key" used as reference.

Ones that start with "StepMapData" will check if that scene contains that sequence, so "StepMapDataChallenge" will be patched into every scene containing that string, AKA every challenge stage. The longer the key, the more specific you can get.

"Common" is always patched in, and the other versions of Common are patched if the level contains at least one entity of that category.

Some objects change based on Tileset, such as pop flowers and arrow signs, so for my own convenience, Those variants of those objects are always patched in when using those specified tilesets. I'll be implementing a cleaner solution later.
